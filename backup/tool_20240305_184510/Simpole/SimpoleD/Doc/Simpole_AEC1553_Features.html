AEC1553 Features（AEC1553 特征）

1553总线系统由总线控制器（Bus Controller，BC）、远程终端（Remote Terminal，RT）和总线监听器（Bus Monitor，BM）组成，采用双冗余热备份总线Bus A、Bus B， 1553总线系统框架图如图所示。

BC负责总线上数据流向，只有BC才能向数据总线上发送命令字，任意时刻，系统中必须有且只能有一个BC。RT是响应BC发送过来的命令，在总线与子系统之间按照有效的指令规定的方式传输数据或执行模式代码操作，系统中最多有31个RT。BM，也称为MT，只能接收总线上数据，可选择性接收数据总线上的信息并保存，如收集到的信息用于对飞行测试记录或飞行任务的分析，系统中可以无BM。

MIL-STD-1553 总线的帧结构是以命令字、状态字和数据字的形式表现出来的。其中命令字是控制器用于向终端或显示器发送命令信息； 状态字则是终端向控制器发送关于总线和终端相连子系统等方面的状态信息；而数据字则用于控制器和终端之间、终端与终端之间发送的相关数据信息。1553总线已被卫星、飞船、国际空间站、港口运输、舰船、坦克、火炮等多个行业广泛采用。

板卡特点

该板卡具有以下特点：

  • 32bit/33MHz PMC总线
  • 双通道多功能1553数据通讯卡
  • 每通道1个BC（总线控制器）
  • 每通道0~31个RT（远程终端）
  • 每通道1个MT（总线监视器）
  • BC、RT、MT 可同时存在
  • 双冗余总线
  • 每通道带RTC功能（可选），RTC计数器48位，分辨率可设
  • 32位时标，分辨率1μs
  • 软件可设置应答超时时间
  • 大容量的数据存储：16M × 16bit

一般规格

  • 物理尺寸：150mm×75mm
  • 连接器：J30J-25ZKWP7-T/PMC P4连接器
  • 工作电源：5V
  • 工作温度：常温实验室环境
  • 相对湿度：实验室环境
